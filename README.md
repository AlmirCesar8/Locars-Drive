Projeto Locadora de Veículos - Locars Drive

Esse projeto é desenvolvido pelo grupo Alisa Meu Pelo (AMP), composto pelos integrantes Almir Cesar, Fábio Benjamin e Gabriel Campos, alunos do Instituto Federal de Educação,Ciência e Tecnologia de São Paulo - Campus São Paulo.

O Projeto Locadora de Veículos é um projeto final requisitado pelos professores do Curso Técnico Integrado de Desenvolvimento de Sistemas, sendo lecionado pelas matérias: Desenvolvimento Web I (WEB), Laboratório de Programação I (LBP),
Segurança e Desenvolvimento de Software(SGDS), Banco de Dados (BDD) e Artes Virtuais Digitais (AVDI).
